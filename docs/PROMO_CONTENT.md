## A. 홍보 자료 (Canva) — 한 장 포스터 / 카드뉴스

### 표지
- 로고 + **하늘 웰니스 랩**
- 태그라인: **"인바디 데이터로, 나의 변화를 한눈에"**
- 서브: 개인 PT 회원 전용 체성분 관리 앱

### 핵심 4가지 (아이콘 + 한 줄)
1. 📊 **결과지 한 장이면 끝** — 인바디 결과지 업로드하면 자동 인식, 차트·추이 자동 완성 (수기 입력도 가능)
2. 🧬 **부위별 근육 3D 모델** — 내 팔·다리·몸통 근육을 3D로 시각화
3. 💬 **코치와 1:1 · 챌린지 · 커뮤니티** — 피드백·응원·목표 달성을 한 곳에서
4. 📅 **수업 스케줄 · 회차권 관리** — 남은 시수·일정·시간 요청까지

### 하단 CTA
- **지금 시작하기** → (QR 코드) + 링크
- **홈 화면에 앱 설치**: 접속 → "설치하기"(안드로이드) / 공유 → 홈 화면에 추가(아이폰 Safari)

## B. 사용법 가이드 (Figma) — 단계별 화면 안내

### 0) 앱 설치하기
- 링크 접속 → 하단 **"앱으로 설치"** 팝업
- 안드로이드: **설치하기** 한 번 → 홈 화면 자동 추가
- 아이폰: **Safari** 공유 버튼 → **홈 화면에 추가** → 추가

### 1) 회원가입 · 로그인
- 이메일로 가입 → 프로필(이름·생년월일·성별·연락처) 설정
- 성별에 따라 3D 근육 모델(남/여)이 표시됩니다

### 2) 나의 건강 — 측정 추가
- **결과지로 추가**: 인바디 결과지 사진 업로드 → 자동 인식 → 값 확인 후 저장
- **직접 입력**: 결과지 없이 인바디 수치 수기 입력
- 채운 항목만 저장, 빈 항목은 '기록 없음'

### 3) 나의 건강 — 보기
- **종합 밸런스 · 인바디 점수 · 게이지**(표준 범위 대비)
- **부위별 근육 3D 모델**: 드래그 회전, 부위 탭 → 해당 근육 하이라이트
- **추이 그래프**: 날짜별 변화, 1월 대비가 아닌 실제 날짜 기준
- **공개/비공개**: 지표별로 커뮤니티 공개 여부 설정

### 4) 코치 피드백
- 트레이너가 남긴 **"하늘 코치의 피드백"** 확인 → 댓글로 답장

### 5) 챌린지
- 목표(변화/달성) 설정 → 시작 기준값 선택 → **전체·이번 주 성취도** 확인

### 6) 커뮤니티
- **피드**: 글·사진·좋아요·댓글 (멘션 가능)
- **멤버의 체성분**: 다른 회원이 공개한 기록 둘러보기(이름 검색)

### 7) 채팅
- 좌상단 **"그룹 채팅 ▾"** 탭 → **그룹 / 개인** 선택
- **개인 채팅**: 코치와 1:1 (회원), 회원별 1:1 (코치) — 방 만들 필요 없이 바로 대화
- 안 읽은 메시지는 **빨간 숫자 배지**

### 8) 수업 스케줄
- 주간/월간 캘린더에서 수업 확인 (시간·수업명·잔여 시수·상태)
- **시간 요청**: 코치와 수업 시간 조율 (대화는 개인 채팅에도 기록)
- **회차권**: 남은 시수 자동 차감(출석/당일취소)

### 9) 알림
- 새 소식은 **종 알림** + (권한 허용 시) **진동 푸시**
- 알림을 누르면 해당 화면으로 바로 이동
